df - disk free - показывает список всех файловых систем по именам устройств, сообщает их размер, занятое и свободное пространство и точки монтирования.

top - table of processes - выводит список работающих в системе процессов и информацию о них. По умолчанию в реальном времени сортирует их по нагрузке на процессор.

ps - process status - выводит отчет о работающих процессах.

ifconfig - interface configuration - используется для назначения сетевого адреса, изменения настроек параметров сетевого адаптера и ip протокола. Она используется на этапе загрузки операционной системы для настройки сетевых интерфейсов.

ping - проверка целостности и качества соединений в сетях.

netstat - network statistics - выводит на дисплей состояние TCP-соединений (как входящих, так и выходящих), таблицы маршрутизации, число сетевых интерфейсов и сетевую статистику по протоколам. Основное назначение это поиск сетевых проблем и определение производительности сети.

route - используется для просмотра и управления таблиц сетевых маршрутов.

traceroute - определение маршрутов следования данных в сетях TCP/IP.

hosts - hosts файл содержит информацию об известных хостах в сети. Для каждого хоста должна присутствовать отдельная строка со следующей информацией:

	Интернет адрес
	Официальное имя хоста
	Алиасы

Файл находится по адресу: /etc/hosts.

networks - файл содержит информацию об известных сетях, включая DARPA интернет. Для каждой сети есть следующая информация:

	Официальное название сети
	Номер сети
	Алиса.

Файл находится по адресу: /etc/networks.

host.conf (доступ через убунту) - файл содержит информацию о настройках для резолвера.
